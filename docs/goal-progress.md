## 当前终端契约

- 顶层页面为 `HOME`、`TOOLS`、`RECORDS` 与 `FILES`；内部继续兼容旧 SIGNAL/FILES wire mode。
- 主快照协议 v5；工具快照 v2；独立导航协议 v4；异象生命周期协议 v3；Debug 协议 v2；持久化 schema v5；结局版本 v1。
- HOME 显示唯一当前目标；附近信号不覆盖它。TOOLS 提供六项工具，四组候选统一进入信号工具。RECORDS 聚合全部日志并一次性标记已读。FILES 使用 30/70 列表与正文分栏，初始无默认选择。
- 玩家设置住处或自然取得铁即可绑定；取得铁启动标志场景，绑定后由该场景揭示无法归类的信号层，不再校准固定频段。
- 调谐只在真实候选附近、信号工具打开时可用；峰值由候选稳定派生，误差 ≤2 连续 20 Tick 后由服务端复核完成。
- 双针指南针、情境接收器、双行 LCD 与六态手持终端都有自动资源/布局契约。

## 非阻断发布清单

- 用已获授权的独立 OGG 和更高保真美术替换当前可运行资产时，保持现有 ID、字幕与峰值边界。
