## 已锁定决策
| 决策 | 理由 |
|---|---|
| 玩家可见文字只使用简单、能直接理解的中文 | 避免抽象系统词破坏理解和恐怖气氛 |
| 工具页采用 APP 网格并进入详情页 | 首页保持简洁，同时容纳真正便利功能 |
| 文件页左侧 30%、右侧 70% | 文件选择与阅读同时可见，减少来回切页 |
| 初次进入文件页不自动选择文件 | 右侧明确显示“选取文件来查看” |
| 固定频段和全局调频退场，只保留近场异常调谐 | 四页终端已承担分类与导航；试玩证明固定频段只剩重复门槛，而近场锁定仍是有意义的剧情动作 |
| 不修改稳定 ID 和历史记录含义，除非有迁移方案 | 保护旧存档、测试和多人兼容 |

|---|---:|---|
